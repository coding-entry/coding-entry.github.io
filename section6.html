Раздел 6: Внешние библиотеки

За долгие годы существования программирования для многих языков существуют полезные библиотеки, созданные другими программистами, чтобы упростить написание кода для какой либо сферы.

Внешние библиотеки - дополнительные программы, которые подключаются к вашей и дополняют её своими функциями и другими вещами. Их используют для упрощения, ведь некоторые частоиспользуемые функции не каждый сможет написать самостоятельно.

Рассмотрим библиотеку random.

Эта библиотека вводит команды, связанные с генерацией случайных чисел.

Конечно, Python при его первичной установке уже имеет много библиотек, но многие популярные там отсуствуют. Чтобы установить отсутствующие библиотеки или пакеты, нужно использовать терминал(консоль).

pip install random

Эту команду нужно прописать в консоли, чтобы установить эту библиотеку сначала просто в сам Python (random заменяете на название другой библиотеки; часто создатели оставляют быструю команду для копирования на сайте библиотеки).

Работа с консолью - другой уровень, мы его пропустим.

После успешной установки эта библиотека появится в Python (либо глобальном, то есть установленном на всём компьютере, при установке без дополнительных переходов в консоли, либо в виртуальной среде, то есть копией Python, которая устанавливается в проекте).

Если Python, который вы используете в программе, имеет библиотеку, вы можете её импортировать (подключить) в код.

import random

Теперь в коде вы можете использовать функции random, но для их вызова нужно сначала обратиться к библиотеке по её названию.

import random

a = random.random()

# Функция random() (из библиотеки random) возвращает случайное число в диапозоне от 0 до 1

Если вы уверены, что из библиотеки вам нужны только конкретные функции или классы, вы можете импортировать только их.

from random import random, randint

# Функция randint(a, b) возвращает случайное целое число от a до b (не включая b)

При установке конкретных функций и классов не требуется обращаться к библиотеке через название в коде.

Если вы хотите обращаться к библиотеке как то иначе, то можете изменить её название (это бывает полезно, если библиотека используется крайне часто):

import random as rnd

Теперь к этой библиотеке нужно обращаться с помощью "rnd".

Если вы хотите импортировать всю библиотеку, но не хотите обращений в коде, то можете импортировать все функции сразу, но Python сочтёт эти импорты как импорты отдельных функций, а значит для них не нужны будут обращения.

from random import *
