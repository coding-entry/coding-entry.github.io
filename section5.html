Раздел 5: Функции и аргументы, локальные переменные

Читая предыдущие разделы вы могли много раз встретил слова "функция" и то, что она "возвращает" какое-то значение.

Функция - блок кода, которые может принимать на вход переменные и возвращает в ответ значение, предназначенный для выполнения одной задачи, которая встречается многократно по всей программе.

Например, будет правильным действием создать функцию для вычисления количества людей из набора данных, которым больше 18 лет, при условии, что во всей программе вам часто нужно многократно это высчитывать.

Когда функция возвращает значение, это значит, что на место функции в коде встанет это значение.

Если функция ничего не возвращает, а просто делает какую-то манипуляцию с данными, то она называется процедурой.

Агрументы функции - это те значения, которые принимает функция на вход, и которые становятся переменными внутри неё.

Запись создания функции:

def function_name(a, b, ...): # название функции и в скобках сколько угодно аргументов, каждый со своим названием как переменной

... # блок кода

Примеры:

def print_sum(a, b):

print(a + b)

Правила нейминга так же действуют на называние функций.

Когда мы создаём агрумент для функции (например age_list), то создаётся переменная, которая сначала принимает значение по умолчанию, а затем, когда функция будет вызвана, заменит значение на то, что стоит при вызове.

Вызов функции:

print_sum(1, 5) # аргумент а равен 1, аргумент b равен 5.

# при выполнении выполнит команду print(a + b), тоесть print(1 + 5)

Обратите внимание, что при наличии 2-х или более аргументов, при вызове функции указывать их значение нужно в том же порядке, через запятую. Также поддерживается указание на нужные аргументы с помощью их локальных имен, например: print_sum(b=5, a=1))

Чтобы созданная нами функция возвращала значение, нужно в конце функции (внутри блока) использовать служебное слово return и через пробел поставить значение (число, переменная и т.д.)

Объединим все наши знания и создадим полноценную функцию count_adults(), которая будет считать количество возрастов в полученном на входе списке, которые больше или равны 18 годам.

def count_adults(list)

count = 0

for age in list:

if age >= 18:

count += 1

return count

Обратите внимание, что внутри функции была создана переменная count. Она используется для проведения независимых от внешней программы подсчёта числа людей старше 18 лет. Это локальная переменная!

Итак, после создания этой функции, давайте её применим:

a = count_adults([5,19,62,4,43,25])

print(count_adults([7,50,23]))

Пространство, в котором функции выполняются, можно считать за абсолютно другую программу, не связанную с прошлой. Все созданные внутри функции переменные, в том числе и аргументы, становятся локальными и недоступны для остальной программы. Из примера - я не смогу получить значение count вне функции. Вы также не сможете просто так использовать в функции переменные, которые были созданны вне функции (они являются глобальными).

Представьте, что функция - это тостер. Просто так мы не можем, например, достать из тостера какую-то деталь. И тостер тоже не может, например пожарить тост, который мы в него не клали. Все тосты, которые лежат где-то на столе - это глобальные переменные. Детали тостера - локальные переменные.

Чтобы получить доступ к глобальным переменным в функции, нужно использовать служебное слово global, а затем название переменной. Это нужно сделать перед тем, как эта переменная где либо используется в функции, иначе программа захочет создать новую (локальную) переменную с таким же названием (такое возможно: названия глобальных и локальных переменных могут совпадать до тех пор, пока не используются одновременно в функции).

a = 0

def add1():

global a

a += 1

Обратите внимание, что эта функция ничего не возвращает (процедура) и ничего не принимает на вход.

Возвращаясь к примеру с тостером. Когда мы кладём тосты со стола в тостер, тостер получает доступ к изменению глобальных тостов через global. После этого тостер изменяет их, при этом ничего (технически) не возвращая. Тостер просто произвел операции по изменению существующих переменных.

Аргументы функции могут быть опциональными. Если вы захотите, вы можете сделать так, что если аргумент существует, но не указан, то он примет значение по умолчанию, которое вы захотите.

Запись:

def function_name(a = 0, b = "Empty", ...)

...

Ранее упоминались функции print() и input().

Print имеет один обязательный аргумент (то, что будет выведено в консоль) и необязательный (то, что print добавит в конец строки при окончании; имя - end). По умолчанию второй аргумент присваивает значение "\n". Это не просто строка. Она включает в себя специальный символ \n. Он обозначает перенос строки на этом моменте. В случае с print() второй аргумент по умолчанию позволяет автоматически переходить на следующую строчку в консоли после окончания вывода.

print("Hello \n world!", end="") # в конце ничего не добавлять

#Hello

# world!

Input имеет необязательный аргумент - это строка, которая выведется самостоятельно, когда программа начнет ожидать ввода от пользователя.

a = input("Введите своё имя: ")

В конце здесь нет \n !
