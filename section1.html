Раздел 1: Данные, переменные.

Типы данных

Основные типы данных:

Целочисленный (Integer, int) - хранит целые числа.

С плавающей запятой (float) - хранит числа с дробной частью. Если число целое, всё равно предоставляет дробную часть в виде одного нуля после точки.

Строчный (String, str) - является списком символов, текстовой строкой.

Логический/Булевый (Boolean, bool) - может быть либо "истина" (True), либо "ложь" (False). Технически, такие данные могут быть приравнены к 1 и 0 соответственно.

Список (list) - является набором элементов. Как и строчный имеют свой чёткий порядок и могут изменять свой размер. Здесь элементы сами по себе являются такими же данными.

Продвинутые типы данных:

Кортеж (tuple) - набор элементов, имеющий фиксированное количество элементов.

Массив (array) - схож со списком, но включает в себя данные одного типа.

Множество (Set) - беспорядочный набор уникальных элементов.

Словарь (Dictionary, dict) - влючает в себя элементы и ключевые обозначения для каждого, по которым их можно индексировать.

Переменные

Переменная (variable) - это некий объект, хранящийся в памяти компьютера только при выполнении программы, содержащий в себе какие либо данные вышеописанного типа.

Во многих языках программирования тип переменной (тип данных) задаётся единожды при её создании (объявлении). В других - тип может быть изменен в процессе выполнения.

У каждой переменной должно быть название. Есть строгие ограничения:

- Название не должно начинаться с цифр и других специальных символов, а также не должно содержать специальные символы, которые могут быть неправильно сочтены компьютером, например "+", "-" и пробел (вместо пробела можно использовать нижнее подчёркивание "_");

- Название должно быть уникальным в своей среде применения (глобально или локально).

Также есть рекомендации нейминга переменных, которые не обязательны, но помогают программисту(-ам) удобно и чётко "читать" код. Например:

- Банальное правило - называть переменные включая их смысл в коде. Не стоит называть переменные по типу "something123", если эта переменная имеет свою цель в программе. Лучше будет назвать переменную "people_count", если эта переменная хранит в себе численное количество людей.

- Стремитесь укорачивать название, не теряя основного смысла. Например: не стоит называть переменную "people_counter_number_variable", а стоит назвать просто "people_count". Также не стоит слишком укарачивать названия.

- Сохраняйте одинаковый стиль названий. Стиль включает в себя то, как вы разделяете несколько слов в названии. Например, если вы ранее назвали переменную "people_count" (разделение слов с помощью нижнего подчёркивания), не стоит называть следующую переменную "humanAge" (разделение слов с помощью изменения регистра).

- Придерживайтесь корректных английских названий.

- Сохраняйте обозначение одного и того же одинаковыми словами. Например, если вы раннее назвали переменную "trip_cost", то не стоит называть следующую переменную "travel_duration", в случае, если речь идет об одном и том же - путешествии.

Объявление переменной (в Python)

Чтобы создать (объявить) переменную (для примера будем называть переменные простыми буквами) нужно сначала написать название, а затем, через знак "=" присвоить ей какое либо значение.

Например:

a = 15

b = "Hello"

Пробел тоже является символом

В разных языках программирования объявление переменных отличается. В Python, например, не требуется указывать конкретный тип данных заранее и отдельно от значения.

Возвращаясь к теме типов данных, для их обозначения в языке программирования есть свои правила:

- Числа (int, float) пишутся обычными числами. Для отделения дробной части всегда используется точка, а не запятая. 1.05, -2

- Строки (str) всегда с двух сторон ограничиваются ковычками. " или ' , причем с двух сторон должны быть одинакового типа ковычки. "Hello world!"

- Списки (list) ограничиваются с двух сторон квадратными скобками [ ], и при перечислении элементов они отделяются запятой. [a, b, c, d]

Для продвинутых типов данных:

Кортеж: (a, b, c)

Словарь: {"word": a, "word2": b}

Множество: {a, b, c, d}

В любой момент кода переменная может быть перезаписана. Например:

a = 1

a = 4

В Python также можно изменять тип данных при перезаписи.
