Раздел 2: Обращение к переменным и операции над ними

Переменные были бы бесполезными, если бы их можно было только создавать. Их предназначение - использоваться в вычислениях, при этом сохраняя возможность принимать другие значения, в зависимости от разных действий.

К переменным можно обратиться с помощью их названия в любом вычислении, в любом моменте, когда часть программы принимает какое-то значение. При этом внутри компьютера в этот момент в то место, где было обращение к переменной, подставляется актуальное её значение. Например, написав внутри программы people_count, это будет означать, что при выполнении вычисления с этим обозначением автоматически подставится значение, имеющая переменная people_count на момент выполнения.

Математические операции:

+, -, *(умножить), /(разделить с дробной частью), //(разделить нацело), ** (возвести в степень), % (получить остаток от деления)

Давайте посмотрим на примеры базовых манипуляций переменными:

a = 1

b = a + 1

Значение "b" будет равно 2.

Если бы строчка "b = a+1" стояла где-то далеко, после вычислений, после которых, например, "a" приняло бы значение 10, то "b" станет 11.

a = 1

b = 2

c = a + b

В этом случае "c" будет равно 3.

Обратите внимание, что практически все изменения значения переменной включают в себя перезапись этой переменной. Например, если ранее "a = 1", то в будущем "a = a + 1" перезапишет "a", при этом используя старое значение "a".

Для математических операций, касающиеся одной и той же переменной, имеются сокращения. Например:

Вместо "a = a+1" пишется "a += 1". Таким же образом и с другими знаками математических операций.

В Python разрешается применять некоторые математические операции к строчным и булевым данным, а также спискам. Например:

a = "Hello" + "world!" # "Helloworld!"

b = "Hi! " * 3 # "Hi! Hi! Hi!"

c = [1,2,3] + [0] # [1,2,3,0]

Серым текстом (комментариями) показаны результаты. Такие вставки в коде не влияют на её выполнение.

В ряде языков программирования данные можно на лету преобразовывать в другие типы данных. Например, можно из числа 15 создать строку "15". В Python это делается так:

a = str(15) # 15 в целочисленном типе данных

b = int("6") # 6 в целочисленном типе данных

c = str(b) # "6" в строчном типе данных

Учтите, что данные должны быть объективно понятны для превращения, иначе будет ошибка. Нельзя, например, написать "a = int("Hello")"

Здесь, чтобы преобразовать в другой тип данных используются названия этих данных, причем в сокращённом виде. Говоря на будущее, эти слова, вместе со скобками, охватывающие первичные данные, являются функциями, которые принимают на вход данные, и возвращают преобразованные. Подробнее об этом в разделе 5.

Изменение и свойства списков

Чтобы обратиться к элементу списка, а не ко всему списку, используется индексирование. Можно получить элемент из списка "a" под номером "n", написав: a[n]. Учтите, что порядковые номера элементов (индексы) начинают нумерацию с нуля! 0, 1, 2, 3...

Дополнительно: в качестве индекса могут использоваться отрицательные числа. Тогда отсчёт будет вестись с конца списка к началу, и самый последний элемент списка будет иметь индекс -1.

a = [1,5,10]

b = a[0] # b = 1

c = a[-1] # c = 10

Такая же индексация работает для строчных данных.

В отличии от других типов, списки можно изменять, не перезаписывая полностью. Это выполняется с помощью комманд (методов) append(), remove(), pop(), insert().

Эти методы, хоть и являются функциями, о чем вы узнаете позже, но вызываются они по-другому, потому что они привязаны конкретно к переменной. Сначала нужно обратиться к списку или похожей переменной, а затем через точку написать метод, который вы хотите использовать.

a = [1,2,3]

a.append(4) # - добавляет элемент (4), указанный в скобках, в конец списка

a.insert(0, -1) # - вставляет элемент (-1) под индексом (0)

a.remove(2) # - ищет элемент (2) и удаляет

a.pop(1) # - удаляет элемент, который находится под индексом (1)

Нередко вам понадобится получить длину списка (или другого набора данных). Тогда вы можете вызвать функцию len(a), где "a" - это ваш набор данных. Учтите, что len() может работать не только со списками, но и например с строчными данными.

a = "Hello"

b = len(a)

Тогда "b" будет равно 5

Крайне частоиспользуемая функция (о них позже) - print(). Эта функция позволяет вывести строку или значение переменной в консоль Python. Дополнительные детали о print() - в разделе 5.

print("Hi")

print(a)

Также часто встречается функция input() в программах начинающих. Эта функция позволяет задать значение переменной, которое было написано пользователем в консоли. Программа остановится, когда увидит input() и будет ждать, пока пользователь напишет что-то в консоль.

a = input()
