Раздел 4: Циклы

Очень часто программы должны выполнять части кода многократно, один за другим, и часто нам неизвестно заранее, сколько раз нужно будет это сделать. Для таких ситуаций существуют циклы. Цикл - часть кода, которая повторяется некоторое количество раз (не обязательно с одними и теми же исходами) и ограничена каким то условием.

В Python и во многих других языках используются два основных типа циклов: while (выполнять пока...) и for (выполнять, проходя по набору данных и пока он не закончится).

Дополнительная информация: функция range(a, b, c):

Функция range(a, b, c) возвращает (выдаёт на выходе) список, который является арифметической последовательностью от минимального числа (a) до максимального (b) не включая с шагом (c) (опционально, по умолчанию установлено значение 1).

Примеры:

a = range(1, 5) # [1,2,3,4]

Также можно использовать на вход только одно число. Тогда это число станет максимальным для последовательности, а отсчёт начнётся с 0.

b = range(6) # [0,1,2,3,4,5]

Цикл while

Цикл while будет выполнять блок кода пока условие, указанное в начале цикла, является истинным.

while a < 10:

a += 1

Здесь мы многократно добавляем к некой переменной "a" единицу до тех пор, пока "a" не станет больше либо равно 10. После этого цикл окончится и программа продолжит выполнять код, которых находится ниже цикла.

while True: - это вечный цикл, так как булевое значение всегда является истинным. Будьте аккуратны с применением такого цикла!

Цикл for

Цикл for будет выполнять блок кода пока в наборе данных, указанный после служебного слова in, есть элементы для итерации. Итерация имеет два значения (для всех типов циклов). 1 - Итерацией называется одно повторение блока кода циклом (Например "цикл произвёл две итерации"). 2 - Итерация (по чему-либо) - "прохождение" по набору данных, выдавая каждый элемент поочерёдно. Для цикла for необходима более сложная запись, имеющая создание новой переменной для записи в нее элементов из итерации набора данных. Очень часто такая переменная имеет название "i".

for i in [0,2,3,6]:

print(i)

Здесь цикл повториться 4 раза (для каждого элемента в списке), и выведет сначала 0, потом 2, потом 3, потом 6.

Удобно в применении здесь функция range(), описанная выше. С её помощью можно легко поочередно присваивать переменной итерации последовательные числа или, если вам эти числа не нужны, можно точно указать, сколько раз повторить цикл.

for i in range(10)

...

Если вы не используете переменную итерации в каких либо вычислениях, вы можете назвать эту переменную "_" в качестве пустой переменной.

Обратите внимание! Если у вас есть несколько циклов, вложенных один в другой (такое разрешено и часто используется), то не рекомендуется называть переменные итерации одинаково, если они используются для вычислений в программе.

Также эти переменные не рекомендуется использовать вне циклов.

Прерывание и пропуск

Внутри циклов можно использовать служебные слова break и continue для преждевременного полного прерывания цикла и пропуска одной итерации соответственно.

while True:

a += 1

if a >= 10:

break # прерывание цикла когда a станет больше или равно10

for i in age_list:

if i < 18:

continue # пропуск итерации, где i меньше 18

else:

adult_count += 1

Когда срабатывает continue программа игнорирует все дальнейшие команды пока не дойдет до конца блока, затем вернется в начало и продолжит итерацию (при наличии) или просто повторение блока.

Когда срабатывает break программа полностью завершает цикл, игнорирует дальнейшие команды в блоке цикла, и когда доходит до конца не возвращается в начало, а начинает выполнять команды после цикла.
