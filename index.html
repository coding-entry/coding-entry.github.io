Знакомство

Если вы хотите писать программы, вы должны понимать, как компьютер читает ваш код и переводит его на свой язык.

Самое главное - компьютер читает программу построчно (если точнее, то посимвольно). Одна за другой, сверху вниз, никак иначе. Поэтому порядок создавания вами чего-либо в программе так важен!

Все примеры программирования будут использовать язык программирования Python. Он является наиболее понятным для новичков. Другие языки могут иметь свои особенности (например, во многих языках программирования строки отделяются не только переносом строки (для читаемости), но и знаком ";", который является символом того, что "строка" окончена, и компьютеру стоит начать читать следующие символы как следующую), иметь разное написание одних и тех же функций (синтаксис), но основополагающие компоненты остаются более менее неизменными.

Не стоит забывать и о огромном сообществе программистов на разных языках (как языках программирования, так и национальных) и множестве способов найти решение появившейся проблемы.

Все термины в программировании, команды и явления обозначаются английскими словами. Поэтому знание его очень важно! Вам не только будет легче запоминать и вспоминать изученные вещи, но еще вы сможете предугадывать, что делает неизвестная вам команда, по её названию, сможете читать официальную документацию для языков программирования или библиотек (об этом в разделе 6), сможете более эффективно искать решения в интернете.

Установка Python

Все языки программирования необходимо сначала установить на устройство, чтобы компьютер знал, как переводить код с этого языка на свой.

Процесс установки для большинства языков не сложен.

Чтобы установить Python на свой компьютер, перейдите по этой ссылке (https://www.python.org/downloads/) и загрузите установщик нужной вам версии (рекомендуется использовать самую последнюю). После этого запустите установщик и, следуя инструкциям в нём, завершите установку. Теперь ваш компьютер знает, что такое Python и как его читать!

Этого недостаточно, чтобы полноценно начать программировать. Вам нужно место, где вы сможете писать, редактировать, открывать и сохранять ваш код. Программы, позволяющие это делать, называются IDE (Integrated Development Environment) (среда разработки).

Когда вы установили Python, на компьютере также автоматически появилась консоль Python, а также стандартная IDE. Хоть эта IDE вполне подходит для программирования, но она проигрывает во многих факторах другим.

Любой код сохраняется в отдельном файле с расширением языка программирования (для python - .py). Технически, любой файл программы - это просто текстовый файл. Если сильно захотеть, можно писать программы в стандартном блокноте, а затем вручную заменить расширение на необходимое.

IDE для Python

IDE из себя представляют многофункциональные программы, которые приспособлены к удобному редактированию и запуску кода. Сейчас почти все IDE поддерживают множество популярных и не очень языков программирования (при установке либо через дополнительные пакеты). Есть IDE более специализирующиеся на одних языках, есть всенаправленные.

Я рекомендую использовать IDE под названием PyCharm. Она специализируется на Python и на его библиотеках, бесплатна и имеет большой функционал.

Подробную инструкцию по установке я давать не буду.

Во многих IDE, в том числе PyCharm, присутствует система проектов. Проект - это просто папка, которая включает в себя файлы кода, а также некоторые скрытые данные для IDE. Если ваша IDE не требует создание проектов, просто откройте файл с вашим кодом и начинайте редактирование. Если же требует, на первых порах можете создать один проект, и создавать в нём все ваши файлы. На первых шагах вам не потребуется создавать несколько файлов для одной программы.
